# 📦Inventory-Management-System-using-Cpp
Developed during my C++ Programming Internship, this console-based Inventory Management System manages product records efficiently. Features include Add, Display, Search by ID/Name, Update Quantity, Delete Item, and Low Stock Alerts using C++ concepts like structures, arrays, loops, and switch-case.


# 📌 Project Overview

The Inventory Management System allows users to:
- Add inventory items
- Display available items
- Search items by ID or Name
- Update item quantity
- Delete items from inventory
- Monitor low stock items

The project demonstrates the use of:
- Structures
- Arrays
- Loops
- Conditional Statements
- Functions of C++ STL (`string`)


# ✨ Features

## 1️⃣ Add Items
Users can add new items to the inventory by entering:
- Item ID
- Item Name
- Quantity


## 2️⃣ Display Items
Displays all inventory items in a tabular format including:
- Item ID
- Item Name
- Quantity
Also shows a **Low Stock Alert** when quantity is less than 5.


## 3️⃣ Search Items by ID
Allows users to search for an item using its unique Item ID.


## 4️⃣ Search Items by Name
Allows users to search inventory items using the item name.


## 5️⃣ Update Quantity
Users can update the quantity of an existing inventory item.


## 6️⃣ Delete Item
Allows users to remove an item from the inventory using Item ID.


## 7️⃣ Exit
Exits the application safely.

# ▶️ How to Run the Program

## Step 1: Save the File

Save the source code as:

```bash
inventory.cpp
```


## Step 2: Compile the Program

Using G++ compiler:

```bash
g++ inventory.cpp -o inventory
```


## Step 3: Run the Program

### For Windows

```bash
.\inventory
```

### For Linux/macOS

```bash
./inventory
```
